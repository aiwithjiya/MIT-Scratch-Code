# 🏙️ City Helicopter Scratch Project

## 📌 Overview

This Scratch project shows a lively city scene with buildings, roads, cars, and a helicopter on top of a building. The goal is to create an animated environment that looks realistic and interactive.

---

## 🎯 Project Objective

* To design a city environment using Scratch
* To animate objects like cars and helicopter
* To learn basic Scratch concepts like sprites, motion, and events

---

## 🧱 Components Used

* **Sprites:**

  * Buildings
  * Helicopter
  * Cars
  * Trees and street objects
* **Backdrop:**

  * City skyline background

---

## ⚙️ How It Works 

### 1. Setup the Background

* Choose or create a city backdrop
* Add buildings and arrange them properly

### 2. Add Sprites

* Insert helicopter sprite and place it on top of a building
* Add car sprites on the road
* Include trees, lamps, and other decorative elements

### 3. Helicopter Animation

* Use "when green flag clicked"
* Add motion blocks to slightly move helicopter (up/down or hover effect)

### 4. Car Movement

* Make cars move from left to right
* Use "forever" loop for continuous movement
* Reset position when reaching the edge

### 5. Add Details

* Adjust sizes and positions of sprites
* Add small animations (like blinking lights or moving objects)

---

## ▶️ How to Run

1. Open the project in Scratch
2. Click the **green flag**
3. Watch the city come alive with animations

---

## 💡 Learning Outcomes

* Understanding of Scratch interface
* Basic animation techniques
* Use of loops and motion blocks
* Scene design and creativity
